# Custom Agent Rules for Cohort 3.0 Workspace

## Rule: Maintain and Utilize the Knowledge Base
Whenever a new conversation starts, or before answering any coding or instructional question in this workspace, you MUST read the `knowledge_base.md` file located at the root of the workspace (`d:\Learning\Cohort 3.0\knowledge_base.md`). 

This file tracks the user's progress, what concepts they have mastered, and their current learning goals. You must use this context to tailor your explanations (e.g., if they haven't learned Flexbox, do not assume they know it).

Furthermore, after you successfully help the user implement a new feature, learn a new concept, or debug a major issue, you MUST update `knowledge_base.md` to reflect this new progress before concluding your help session. This ensures context is permanently saved for future sessions.
